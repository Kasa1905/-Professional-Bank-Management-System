## Installation Instructions

1. Install Python 3.8 or higher
2. Install MySQL Server 8.0 or higher
3. Run: pip install -r requirements.txt
4. Update config.py with your database credentials
5. Execute database/schema.sql in MySQL
6. Execute database/sample_data.sql in MySQL
7. Run: python main.py

## System Requirements

- Python 3.8+
- MySQL 8.0+
- 4GB RAM minimum
- 500MB disk space
- Windows 10/macOS 10.15/Ubuntu 18.04 or later
